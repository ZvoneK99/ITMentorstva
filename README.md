# ITMentorstva React Course

This repository is the **main repository for the ITMentorstva React course**.  
It contains all projects and exercises for learning **React from scratch**.

All examples focus on **React fundamentals**, including components, state, props, and list rendering.

---

## Repository Overview

This repository includes all exercises and mini-projects created during the ITMentorstva React course.  
The focus is on **practical learning**, covering:

- Functional components  
- State and props  
- List rendering using `map`  
- Event handling (click, input, form submit)  
- React Router for navigation between pages  
- Basic CSS and Bootstrap styling  

---

## Project Structure

- `src/Components/` → React components (e.g., `AllVideos`, `ShowOneVideo`)  
- `src/Pages/` → Pages rendered via React Router  
- `src/videos.json` → Example data for video projects  
- `public/` → Static assets (favicon, index.html, etc.)
